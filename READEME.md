# Analog Clock :clock130:

## Features

- Real-time hour, minute and seconds clock hands
- Glass-morphism styling with layered gradients and inner glow
- Soft highlight "shine" for a polished look
- Fully responsive
- Lightweighted and framework-free

## Tech Stack

- HTML for structure
- CSS (custom properties, gradients, shadows)
- JavaScript for timekeeping & DOM generation
